Instructions

Compiled using Harp:

npm install -g harp

Serve from the _harp directory (for testing):

cd _harp
harp server

Compile for deployment:

harp compile _harp ./

The development pattern has been to make the necessary edits to the source files and followed by a corresponding git commit, then compile and make another commit for the build (see history).

Structure

The design patterns and purposes of different files should be clear after reading Harp documentation. Files beginning with an underscore _ are not directly served but used in other files.

The site uses Bootstrap as its framework and makes use of pug (previously known as Jade), some Markdown for longer text and Less for any custom styling.
